### (Attach)Start Flow:

* Entry (Demo): /doc.php
 - Setting
 - require(...'/root/run/_init.php'); // Init File

* Init File: _init.php
 - //Include path
 - include(RUN_INIT.'/code/cfgs/boot/_paths.php');
 - //Boot file
 - include(RUN_INIT.'/code/cfgs/boot/bootstart.php');

* Boot file: bootstart.php
 - $_cbase Init
 - Load configs, system functions
 - (judge outer scope) `basEnv` reset Environmental 

* Back to Entry file:
 - Run code like this: `$vop = new vopShow();`
 - Show pages
